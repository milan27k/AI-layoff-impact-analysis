# 🤖 AI Layoff Impact Analysis

## 📌 Project Overview

This project analyzes the impact of Artificial Intelligence on workforce reduction across different companies and industries. Using **Python (Pandas) and Power BI**, the project uncovers trends, patterns, and key insights related to layoffs.

## 🎯 Objectives

* Analyze workforce reduction trends across companies
* Identify region most affected by AI-driven layoffs
* Understand company-wise and sector-wise impact
* Perform data cleaning and exploratory data analysis (EDA)
* Build an interactive dashboard for visualization

## 🛠️ Tools & Technologies

* **Python (Pandas)** – Data cleaning & EDA
* **Power BI** – Dashboard visualization
* **Jupyter Notebook** – Analysis workflow

## 📂 Project Structure

ai-layoff-impact-analysis/
│
├── ai_layoff_data_cleaning_eda.ipynb   # Data cleaning + EDA
├── ai_layoff_dashboard.pbix           # Power BI dashboard
└── README.md                         # Project documentation

## 📊 Key Analysis Performed

* Data cleaning (handling missing values, duplicates)
* Company-wise layoff analysis
* region-wise impact analysis
* Year-wise trend analysis
* Average workforce reduction calculation
* High vs Low impact classification

## 📈 Key Insights

* The **Enterprise sector** is the most affected by AI-driven layoffs
* Layoffs are led by major companies like Amazon and Oracle
* Workforce reduction shows an increasing trend over time
* A small number of companies contribute a large share of total layoffs
* Certain industries show significantly higher average layoffs

## 📊 Dashboard Highlights

* KPI cards: Total layoffs, average reduction
* Company-wise and region-wise analysis
* Time trend visualization
* Interactive filters for deeper insights

## 🚀 How to Use

1. Open the Jupyter Notebook for data analysis
2. Run the notebook to reproduce EDA
3. Open the Power BI file to explore the dashboard
4. Interact with filters to gain insights

## 💡 Conclusion

This project shows how AI is impacting jobs by increasing layoffs across different companies and industries. It helps understand which sectors are most affected and how workforce trends are changing over time.
